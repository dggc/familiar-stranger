\section{Conclusão}
\frame{
    \frametitle{Conclusão}

    Avaliou-se o impacto da utilização da um técnica de codificação para uma rede móvel real.
    \vspace{.4cm}
    \begin{itemize}
     \item \textbf{Epidêmico}: Apresentou uma performance razoável.\\
     $\rightarrow$ Menos que 20\% dos pacotes não foram entregues \\
     $\rightarrow$ 60\% dos pacotes foram entregues com $10^3$ rounds \\
     \vspace{.4cm}
     \item \textbf{SocialRoute}: Não apresentou bons resultados\\
     $\rightarrow$ Nenhum pacote foi entregue com sucesso.\\
     $\rightarrow$ Acredita-se que a proximidade física entre amigos foi insuficiente.\\
    \end{itemize}


}
